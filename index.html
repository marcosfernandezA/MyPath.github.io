🧬 XSS (Cross-Site Scripting)

XSS es una vulnerabilidad que permite a un atacante inyectar scripts maliciosos en páginas web vistas por otros usuarios. Esto puede llevar al robo de información, manipulación de la página y otros ataques.

💾 SQL Injection

SQL Injection es una vulnerabilidad que permite a un atacante manipular las consultas SQL de una aplicación, accediendo o modificando datos de manera no autorizada.
